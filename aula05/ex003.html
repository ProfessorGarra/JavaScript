Testando o JS

  • + concatena e adiciona.
  • para adicionar precisamos number + number.
  • para concatenar precisamos de string + string.
  • Number.parseFloat() é usado para converter qualquer string números reais.
  • Number.parseInt() é usado para converter qualquer string números inteiros.
  • Number() é usado para converter qualquer string em número.
  • Para converter de número pra string, usamos String() ou usamos n.toString()
  • Template string inicia com crase ´,segue por ${} e encerra com crase ` e ajuda a concatenar melhor os comandos.
  • .toFixed(x) expande as casas decimais de um número onde x é a quantidade de casas decimais
  • .toLocaleString("pt-br",{style:"currency", currency:"BRL"}) define o valor monetário, onde BRL pode ser trocado po USD e por EUR
  • .toFixed(2).replace('.',',') troca o ponto por virgula
